1. Clone this repo using `git clone https://github.com/MetrixIT-UrbanSattva/USCustSrvc.ItemsDetails.git`
2. Move to the appropriate directory: `cd USCustSrvc.ItemsDetails`.<br />
3. Run `yarn` or `npm install` to install dependencies.<br />
4. Run `npm run start` to see the example app at `http://localhost:3104`.
